Normas de Convivencia - SudaPlay

SudaPlay es una plataforma para compartir y disfrutar videojuegos creados por la comunidad. Aunque actualmente no existe sistema de comentarios, todos los usuarios deben respetar las siguientes normas al usar la plataforma.

1. Respeto y uso adecuado del perfil

No se permite usar nombres ofensivos o inapropiados.

No se permite utilizar imagenes de perfil con contenido sexual explicito, violento o discriminatorio.

No esta permitido suplantar la identidad de otra persona o marca.

Cada usuario es responsable de la informacion que muestra en su perfil.

2. Conducta en la plataforma

No se permite intentar vulnerar la seguridad del sistema.

No se permite manipular el funcionamiento de los juegos o del sitio.

Esta prohibido el uso de bots o automatizaciones para alterar estadisticas.

3. Contenido inapropiado

No se permite subir juegos que contengan:

Pornografia o contenido sexual explicito.

Violencia extrema o gore explicito.

Discurso de odio o discriminacion.

Contenido que promueva actividades ilegales.

La plataforma se reserva el derecho de eliminar contenido que considere inapropiado.

4. Medidas disciplinarias

El incumplimiento de estas normas puede resultar en:

Eliminacion del juego.

Suspension temporal de la cuenta.

Eliminacion permanente del usuario.
